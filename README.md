
# React Sorcerer Take-Home Test

A Rich Text editor using Draft.js within ReactJS


## Tech Stack

React, Draft.js


## Prerequisite 

Node verision 16 or greater required


## Run Locally

Go to the project directory

```bash
  cd sorcerer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Build for production

```bash
  npm run build
```
